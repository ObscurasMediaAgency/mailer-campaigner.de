Schnellstart

Mit diesen einfachen Schritten startest du in wenigen Minuten deine erste Kampagne.

1. Installation

# Virtuelle Umgebung erstellen
python -m venv venv
source venv/bin/activate

# Abhängigkeiten installieren
pip install -r requirements.txt

# Konfiguration
cp .env.example .env
nano .env  # SMTP-Daten eintragen

2. SMTP-Konfiguration

Trage deine SMTP-Zugangsdaten in die .env-Datei ein:

export SMTP_HOST="smtp.deinserver.de"
export SMTP_PORT="465"
export SMTP_USER="mail@deine-domain.de"
export SMTP_PASS="dein-sicheres-passwort"

3. Erste Kampagne

Teste deine Konfiguration mit einem Probelauf:

# Kampagnen anzeigen
python send_campaign.py --list

# Vorschau erstellen
python send_campaign.py meine_kampagne --preview

# Test-Mail senden
python send_campaign.py meine_kampagne --test deine@email.de

Kampagnen erstellen

Jede Kampagne besteht aus einer YAML-Konfigurationsdatei und einer Kontaktliste.

Kampagnen-Struktur

campaigns/
├── meine_kampagne/
│   ├── campaign.yaml    # Konfiguration & Inhalt
│   ├── contacts.csv     # Kontaktliste
│   └── preview.html     # Generierte Vorschau

campaign.yaml Beispiel

name: "Meine Kampagne"
subject: "Betreffzeile der E-Mail"
contacts: "contacts.csv"

content:
  greeting: "Sehr geehrte Damen und Herren,"
  intro: |
    Einleitungstext mit Link zur 
    Website.
  body:
    - "Erster Absatz des Haupttexts."
    - "Zweiter Absatz mit Details."
  cta:
    text: "Jetzt kontaktieren →"
    url: "https://example.com/kontakt"

contacts.csv Format

FIRMA,DOMAIN,EMAIL,PROBLEM
Musterfirma GmbH,https://musterfirma.de,kontakt@musterfirma.de,"Problem-Beschreibung"

SMTP-Konfiguration

Konfiguriere deinen E-Mail-Server für zuverlässigen Versand.

Umgebungsvariablen (.env)

export SMTP_HOST="smtp.deinserver.de"
export SMTP_PORT="465"
export SMTP_USER="mail@deine-domain.de"
export SMTP_PASS="dein-sicheres-passwort"

Absender-Daten (config/sender.yaml)

sender:
  name: "Max Mustermann"
  title: "Gründer & Entwickler"
  email: "mail@deine-domain.de"

company:
  name: "Deine Firma"
  domain: "deine-domain.de"
  url: "https://deine-domain.de"

rate_limit:
  delay_seconds: 80  # ~45 Mails/Stunde

Empfohlene SMTP-Einstellungen

  • Port 465: SSL-verschlüsselt (empfohlen)
  • Port 587: STARTTLS-verschlüsselt
  • Rate-Limit: 45-60 Mails/Stunde vermeidet Spam-Flags
