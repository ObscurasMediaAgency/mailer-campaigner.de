Allgemeine Hinweise

Diese Website erhebt und verarbeitet keine personenbezogenen Daten auf externen Servern. Alle Einstellungen und Präferenzen werden ausschließlich lokal in Ihrem Browser (localStorage) gespeichert und verlassen Ihr Gerät nicht.

Hosting

Diese Website wird bei einem externen Hosting-Anbieter gehostet. Beim Besuch der Website erfasst der Server automatisch technische Informationen in sogenannten Server-Log-Dateien, die Ihr Browser übermittelt:

  • IP-Adresse (anonymisiert)
  • Datum und Uhrzeit der Anfrage
  • Angeforderte Seite/Datei
  • Browser und Betriebssystem
  • Referrer-URL (vorher besuchte Seite)

Diese Daten werden zur Sicherstellung des Betriebs und zur Abwehr von Angriffen benötigt und nach spätestens 7 Tagen automatisch gelöscht.

Mailer Campaigner Software

Die Mailer Campaigner Software ist eine lokale Desktop-Anwendung, die ausschließlich auf Ihrem Computer läuft. Bezüglich Datenschutz gilt:

  • Alle Daten (Kampagnen, Kontakte, Templates) werden lokal auf Ihrem System gespeichert
  • Es findet keine automatische Cloud-Synchronisation statt
  • Es werden keine Nutzungsstatistiken an uns übermittelt
  • E-Mails werden direkt über Ihren eigenen SMTP-Server versendet

Wichtig: Sie sind als Nutzer der Software selbst für die datenschutzkonforme Verwendung verantwortlich, insbesondere beim E-Mail-Versand an Geschäftskontakte. Beachten Sie die geltenden Gesetze (DSGVO, UWG).

SSL-/TLS-Verschlüsselung

Diese Website nutzt aus Sicherheitsgründen eine SSL- bzw. TLS-Verschlüsselung. Eine verschlüsselte Verbindung erkennen Sie daran, dass die Adresszeile des Browsers von „http://" auf „https://" wechselt und an dem Schloss-Symbol.

Wenn die SSL- bzw. TLS-Verschlüsselung aktiviert ist, können die Daten, die Sie an uns übermitteln, nicht von Dritten mitgelesen werden.
